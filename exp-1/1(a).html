This page shows different types of lists: ordered, unordered, nested lists, an ordered list inside an unordered list, and a definition list.

1. Ordered list (<ol>)

Example HTML:

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

Rendered output:

  1. First item
  2. Second item
  3. Third item

2. Unordered list (<ul>)

Example HTML:

<ul>
  <li>Apples</li>
  <li>Bananas</li>
  <li>Cherries</li>
</ul>

Rendered output:

  • Apples
  • Bananas
  • Cherries

3. Nested lists (mixing <ul> and <ol>)

Example HTML (a nested unordered list inside an ordered list):

<ol>
  <li>Fruits
    <ul>
      <li>Citrus
