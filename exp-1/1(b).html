HTML Hyperlinks using <a> Tag

The <a> tag creates hyperlinks. The two most important attributes are:

  • href: The destination URL or anchor ID.
  • target: Where to open the link (e.g., _self, _blank).

1. Link opening in the same tab (default)

<a href="https://www.example.com">Visit Example (same tab)</a>

Rendered output: Visit Example (same tab)

2. Link opening in a new tab/window

<a href="https://www.wikipedia.org" target="_blank">Open Wikipedia in new tab</a>

Rendered output: Open Wikipedia in new tab

3. Link to a specific section on the same page

<a href="#contact">Go to Contact Section</a>

Rendered output: Go to Contact Section
